== Description ==

Connect Contact Form 7 to Telegram is a Telegram notification plugin for WordPress that makes it really simple to send your Contact Form 7 forms directly to your Telegram account.

This plugin helps your users to send their data to your telegram account after the contact form submitted and you can recieve it anytime and anywhere.

You can choose to open telegram window in either current window or new window.

<strong>Connect Contact Form 7 to Telegram</strong> allows you to send WordPress contact form leads directly to your Telegram account.

To send messages you need to follow several simple steps to configure social accounts.

** Telegram account configuration steps **
1. Create the Telegram Bot and save the Bot Token parameter on the settings page Contact Form 7 â€“> CF7 Telegram.
2. Enable Telegram message in Contact Form 7 â€“> Edit contact form â€“> Go to Telegram tab â€“> Enable status.

Now you will able to add users or group to subscriber list. Subscribers list shows all chats which can receive telegram message.
To add a user send the /cf7tel_start command to your bot.
To add a group add your bot to the group and send the /cf7tel_start command to your group.

== Frequently Asked Questions ==

= How to connect with the Whatsapp account? =

Go to Admin Dashboard â€“> Contact Forms â€“> Edit contact form â€“> Go to WhatsApp tab. Then configure given settings. 

= How to create the Telegram-Bot? =

It is very simple. Please, follow to  [official documentation](https://core.telegram.org/bots#how-do-i-create-a-bot "Telegram docs").

= What is Telegram Chat ID & how to get it? =

The Chat ID parameter is your Telegram-identifier. But this is not your phone number or Telegram-login (@xxxxxxxx). 
You can see your Chat ID by typing anything to Telegram-Bot <code>@cf7telBot</code>.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin configure telegram account and whatsapp account configuration:
1. To send telegram messages go Admin dashboard â€“> CF7 Telegram. After that go to Contact forms â€“> Edit Form â€“> Telegram(tab)
